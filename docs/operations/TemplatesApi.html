This is the API

templatesGet

Gets a list of Template Metadata objects for all templates. Up to 25 results returned before results are paginated.

URL

https://api.dyspatch.io/templates

HTTP Method

GET

Response Type

TemplatesRead

Parameters

  • Header: Accept String

    A version of the API that should be used for the request. For example, to use version \"2018.08\", set the value to \"application/vnd.dyspatch.2018.08+json\"

  • Query: cursor String

    A cursor value used to retrieve a specific page from a paginated result set.

templatesTemplateIdGet

Gets a template object with the matching ID. If the template has published content the \"compiled\" field will contain the template .

URL

https://api.dyspatch.io/templates/{templateId}

HTTP Method

GET

Response Type

TemplateRead

Parameters

  • Path: templateId String

    A template ID

  • Header: Accept String

    A version of the API that should be used for the request. For example, to use version \"2018.08\", set the value to \"application/vnd.dyspatch.2018.08+json\"
