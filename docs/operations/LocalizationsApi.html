localizationsLocalizationIdGet

Returns a specific localization object with a matching ID

URL

https://api.dyspatch.io/localizations/{localizationId}

HTTP Method

GET

Response Type

LocalizationRead

Parameters

  • Path: localizationId String

    A localization ID

  • Header: Accept String

    A version of the API that should be used for the request. For example, to use version \"2018.08\", set the value to \"application/vnd.dyspatch.2018.08+json\"
